## How to Redeem

### Step-by-Step Process

#### Step 1: Browse & Select
```
1. Click on vouchers or merchandise tab
2. Use search bar to find specific items
3. Filter by category or vendor
4. Read item details and point cost
```

#### Step 2: Confirm Details
```
1. Click "Redeem Now" or "Order Now"
2. Review:
   - Item name
   - Point cost
   - Convenience fee (if applicable)
   - Delivery method
3. Click "Proceed"
```

#### Step 3: Verify with OTP
```
1. We'll send a 6-digit code to your email
2. Enter the code in the pop-up
3. You have 10 minutes to verify
4. Can retry 3 times max
5. Click "Verify Code"
```

**Important**: Your points are deducted ONLY after OTP verification (not before).

#### Step 4: Delivery Details
**For Vouchers**:
- Email auto-filled
- No additional action needed
- Ready!

**For Merchandise**:
- Enter shipping address:
  - Full name
  - Phone number
  - Address line 1 & 2
  - City, state, pincode
- All fields except address line 2 are required
- Click "Complete Order"

#### Step 5: Confirmation
```
âœ“ Redemption successful!
- For vouchers: Check email for code
- For merchandise: Track your order
```

## FAQ

### Points & Wallet

**Q: How do I earn points?**
A: Points are earned through company recognition from peers or managers. Also check promotions and milestone achievements.

**Q: Can points expire?**
A: Depends on company policy. Check tenant settings. Default is never expires.

**Q: Can I transfer points to others?**
A: No, points are personal and non-transferable.

**Q: What if I accidentally redeem?**
A: If you haven't verified OTP yet, close the browser. If verified, contact admin for reversal within 24 hours.

### Redemption Process

**Q: How long does OTP take?**
A: Usually < 1 minute. Check spam folder if delayed.

**Q: What if I don't receive OTP?**
A: 
- Check spam/promotions folder
- Check email address in profile
- Click "Resend OTP"
- Contact support if still no OTP

**Q: Can I cancel redemption?**
A: 
- PENDING: Yes, refresh page
- OTP_VERIFIED: Points can be reversed within 24 hours, contact admin
- PROCESSING onwards: No, but can refuse delivery

**Q: Can I redeem for someone else?**
A: No, redemptions are personal. Each user redeems for themselves.

### Vouchers

**Q: When do I get the voucher code?**
A: Usually within 5 minutes of successful redemption. Check email (including spam).

**Q: How long is the voucher valid?**
A: Check the voucher card - typically 365 days. Note the expiry date!

**Q: Can I use partial voucher amount?**
A: It depends on vendor. Most allow, some don't. Check vendor terms.

**Q: Can I get cash back if I don't use it?**
A: No, vouchers cannot be converted to cash or refunded.

**Q: Voucher code not working?**
A:
- Double-check code (copy-paste recommended)
- Ensure you're on correct vendor site
- Check terms and conditions
- Contact vendor support, not Perksu

### Merchandise

**Q: How long to deliver?**
A: Usually 5-7 working days after order confirmed. Holidays may extend this.

**Q: Can I change address after ordering?**
A: If not yet shipped, contact admin immediately with new address.

**Q: What if item is damaged?**
A: Contact support within 24 hours with photos. We'll arrange replacement/refund.

**Q: Can I refuse delivery?**
A: Yes, but points won't be refunded. Better to cancel before shipment.

**Q: Is there return/exchange?**
A: Yes, 7 days from delivery. Contact support for RMA.
